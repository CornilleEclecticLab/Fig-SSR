scripts/eems/README.md

Purpose:  
Run **EEMS** (Estimated Effective Migration Surfaces) to infer and visualize spatial patterns of gene flow and diversity from SSR data.

---

Key files:
- `eems.R` — R script to set up input files, generate params.ini, compile/run EEMS, and create plots with rEEMSplots
- Required inputs:  
  - `genalex_metadata_sp_only.csv` (with longitude/latitude)  
  - `genalex_geno_sp_only.csv` (genotype matrix, 2 columns per locus)  
  - `updated_polygon.outer.txt` (habitat polygon)

How to run:
1. Place input files (`metadata CSV`, `genotype CSV`, `polygon`) in the working directory.
2. Activate the environment:
   ```bash
   mamba activate fig-ssr
   
3.	Run the EEMS setup and workflow in R:
Rscript eems.R
Inputs/outputs:
	Input:
	•	genalex_metadata_sp_only.csv — metadata with coordinates
	•	genalex_geno_sp_only.csv — SSR genotype matrix
	•	updated_polygon.outer.txt — polygon file defining the habitat
	Output:
	•	eems_input/ — contains prepared .outer, .coord, .sites, and params.ini
	•	eems_results/run1/ — results of the EEMS MCMC run
	•	eems_plots/run1/ — migration and diversity maps generated by rEEMSplots
